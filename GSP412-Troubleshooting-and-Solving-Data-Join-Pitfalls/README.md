# GSP412 - Troubleshooting and Solving Data Join Pitfalls

## Overview

This folder contains the companion resources for the **Troubleshooting and Solving Data Join Pitfalls (GSP412)** lab.

In this lab, you'll explore common SQL JOIN issues in BigQuery and learn practical techniques to identify and resolve duplicate records, missing data, and incorrect join results.

## Learning Objectives

After completing this lab, you should understand how to:

- Identify duplicate records
- Work with INNER JOIN
- Work with LEFT JOIN
- Work with RIGHT JOIN
- Work with FULL JOIN
- Understand CROSS JOIN behavior
- Troubleshoot missing records after joins
- Avoid duplicate counting caused by non-unique keys
